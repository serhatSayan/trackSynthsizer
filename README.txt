*Bu program python 3.9.9 x86 venv kullanılarak yapılmıştır, programı çalıştırırken bu virtual environmenti kullanarak çalıştırdığınızdan emin olun.
*Program sıra sıra notaları alıp kare dalgaya dönüştürüp tüm notaları bir array'e atar, daha sonra bu array uygun tipe dönüştürülerek bir wav dosyası olarak kaydedilir. Burada wav dosyasına yazmamızın sebebi parçayı wav dosyasına 2 boyutlu array olarak gönderdiğimizde bunu farklı channel olarak kabul etmesi.
